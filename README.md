* 🧩 HR/General Affairs 실무 경험을 통해 일정 관리, 자산 관리, 보안 프로세스, 협업 커뮤니케이션을 경험했습니다.<br>
* 🚀 AI 면접 서비스, 모의투자 시스템 등 실사용 흐름이 있는 프로젝트를 만들어가고 있습니다.<br>
* ⚡ 개발할 때는 기능 구현뿐 아니라 “왜 이 구조가 필요한지”를 이해하고 설명할 수 있는 개발자를 지향합니다.
